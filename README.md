# Manual Secure Android Login App

Part of a BSc Computer Science dissertation comparing AI-assisted vs manual software development. This is the **manual implementation** of a secure Android login application, featuring SHA-256 password hashing, secure credential storage, and account lockout.

---

## Features
- SHA-256 password hashing (manual implementation)  
- SharedPreferences for credential storage  
- Account creation with input validation  
- Login attempt tracking and automatic lockout after 5 failed attempts  
- Password visibility toggle  
- Clean, intuitive XML-based UI  

---

## Tech Stack
- **Java**  
- **Android Studio**  
- **SHA-256 hashing, SharedPreferences**  
- **XML (UI layout)**  
- **Git**  

---

## Testing
- Password hashing validation  
- Login attempt tracking and lockout checks
